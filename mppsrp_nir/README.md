

# Установка
```
conda create -n mppsrp_nir_env python=3.11
conda activate mppsrp_nir_env
pip install -r requirements.txt
```

# Основная схема использования
генерирование графов -> генерирование синтетических данных под задачу -> 
построение задачи -> использование конкретного решателя задачи -> получение решения -> 
построение путей и плана путей (сортировка по времени) -> подсчет KPI, визуализация
